# 网页版识字游戏 · 交付报告

> 交付日期：2026-07-18
> 形态：单个 HTML 文件（双击即玩、零安装、零后端、可离线）
> 文件：`识字游戏.html`（约 64 KB，含 3000 字拼音库内联）

---

## 1. 产品概述
面向 6 岁儿童的网页识字游戏，主打「有趣」。两条核心玩法：
- **🎯 听音选字测评**：播放汉字读音 → 4 选 1 → 20 题出「识字量估算 + 建议级别」，并解锁对应级别。
- **🎮 抓字小游戏**：汉字从顶部下落 + 听音 → 点中正确字得分/连击，漏接或点错扣 1 命，3 命结束出分。

覆盖《通用规范汉字表》前 3000 常用字，按字频分 30 级（每级 100 字），难度递增。

## 2. 趣味性设计（重点）
- 🦊 字宝吉祥物随表现变表情（😊 普通 / 😎 连击 / 🤩 高连击 / 😢 扣血）
- 答对触发 **Confetti 礼花** + 中文语音夸奖「真棒！厉害！」
- 连击数实时放大显示，分数随连击加成
- 果冻弹性按钮、答对光波动效、暖橙糖果色 UI

## 3. 技术底座
| 项 | 方案 |
|----|------|
| 形态 | 单 `.html`，内联 CSS/JS/数据，零构建零后端 |
| 字库 | 构建期内联 3000 字 `{字:拼音}` 到 `window.DICT`，**离线可用** |
| 发音 | Web Speech API `speechSynthesis`(zh-CN)，**零音频文件**；不支持时降级为视觉提示不崩 |
| 进度 | `localStorage`（最高分 / 最高连击 / 已解锁级别 / 最近测评） |
| 适配 | mobile-first 响应式，按钮 ≥ 44px，桌面端居中 480px 模拟手机 |

## 4. 如何运行
- **最简单**：双击 `识字游戏.html` 用浏览器打开即玩（手机/电脑均可）。
- **手机体验最佳**：用手机浏览器打开（或传到手机后打开），发音与触屏手感最佳。
- **分享**：单文件可直接发给家人/朋友，无需安装任何东西。

## 5. 验收结果（自测）
用 jsdom 加载真实构建产物跑 DOM 流程，**14 项断言全过、0 运行期错误**：
- 首页渲染、localStorage 可读
- 测评 20 题 → 结果页；`highestLevel` / `assessedLevel` 正确写入
- 抓字游戏下落字生成、点中目标字得分增加（0→12）
- 30 个级别格、测评后正确解锁（8~11 级）
- 五视图导航闭环（首页↔测评↔游戏↔结果↔级别）
- 发音在不支持的环境下降级不崩

对照 8 条验收标准（AC01~AC08）：**全部覆盖**。

## 6. 已知限制（非阻塞，MVP 范围）
- 真实设备发音需在「用户手势」内首次触发（iOS Safari 限制）；页面所有发音均在点按后触发，已合规。
- 字宝为 emoji 🦊 占位，可替换为真实插画。
- 纯前端单文件：**无真实后端 / 排行榜 / 付费 / 账号**（按需求范围，未做）。
- 多音字只考常用音（测评/游戏取默认音）。
