## 1. 产品概述

### 1.1 一句话描述

一款面向5-7岁幼小衔接儿童的**微信小程序识字闯关游戏**，以高频常用汉字为内容核心，用图文卡片+冒险闯关玩法让孩子在趣味中识字，用科学复习机制和透明家长面板让家长放心。

### 1.2 目标用户

| 维度 | 描述 |
|------|------|
| 主要使用者 | 5-7岁幼小衔接阶段儿童 |
| 决策者/付费者 | 25-40岁家长（以妈妈为主） |
| 使用场景 | 家庭日常碎片时间（饭后、睡前），单次10-15分钟 |

### 1.3 核心问题

| 问题编号 | 问题描述 | 现状 |
|----------|----------|------|
| P1 | 幼儿园不教识字，小学一年级要求认300字，存在**学习节奏断层** | 家长焦虑，孩子入学后挫败感强 |
| P2 | 现有识字APP**游戏占比过大**，孩子沉迷游戏忽略识字 | 洪恩识字差评：孩子玩了2小时设计衣服游戏 |
| P3 | 现有APP**收费不透明**，自动续费投诉严重 | 洪恩识字被投诉连续扣费57个月 |
| P4 | 主流识字产品均为**独立APP**，需下载安装，门槛高 | 家长不想为试用下载几百MB的APP |
| P5 | 家长**看不到真实学习效果**，无法判断孩子到底学会了没有 | 适趣AI被投诉效果虚标，显示识字1600实际仅认识二三百 |

### 1.4 解决方案

| 方案 | 对应问题 |
|------|----------|
| **微信小程序形态**：免下载、即开即用、分享即传播 | P4 |
| **闯关冒险+图文卡片**核心玩法：每个关卡识字为主、游戏为辅，游戏时长硬性限制 | P1、P2 |
| **透明收费**：免费学50字， thereafter 明码标价一次性买断，无自动续费 | P3 |
| **家长面板**：实时显示已学字、掌握率、薄弱字列表，支持自定义生字本 | P5 |
| **科学复习**：基于艾宾浩斯遗忘曲线自动安排复习，确保真正掌握 | P5 |

### 2.4 市场空白与差异化机会

基于竞品差评分析，识别出以下 **7个市场空白点**：

| 编号 | 市场空白 | 依据 | 差异化机会 |
|------|----------|------|------------|
| **G1** | **微信小程序形态空白** | 主流识字产品均为独立APP，小程序赛道仅有简单工具型产品（花漾识字等），无系统化闯关识字小程序 | 免下载、即开即用、微信社交裂变传播、家长群分享 |
| **G2** | **游戏与学习平衡空白** | 洪恩游戏过多导致沉迷，帮帮动画分散注意力，市场缺少"识字为主、游戏为辅"的平衡型产品 | 闯关模式中游戏环节硬性限时（≤30%时长），核心交互围绕识字本身 |
| **G3** | **透明收费空白** | 自动续费投诉是行业重灾区（洪恩57月、宝宝巴士3年），家长对收费信任度极低 | 免费学50字+一次性买断，永不自动续费，明确公示价格 |
| **G4** | **自定义生字本空白** | 除洪恩有跳字功能外（仍被差评），多数产品按固定顺序教学，家长无法指定学习内容 | 支持家长自定义生字列表（如同步学校课文生字） |
| **G5** | **科学复习机制空白** | 多数产品复习机制简单，适趣AI被投诉"显示识字1600实际仅认识二三百"，效果验证缺失 | 基于遗忘曲线的间隔复习+定期测字验证真实掌握率 |
| **G6** | **亲子共学空白** | 现有产品多让孩子独自玩APP，缺少亲子互动设计 | 设计亲子挑战模式，家长参与闯关，增强互动 |
| **G7** | **学习效果透明化空白** | 家长无法看到真实掌握情况，效果虚标问题严重 | 家长面板展示真实测字结果、掌握率趋势、薄弱字清单 |

## 4. 核心功能列表

### 4.1 优先级与RICE评分说明

- **P0**：MVP必须交付的核心功能
- **P1**：MVP发布后快速迭代的高价值功能
- **P2**：后续迭代的中长期功能

**RICE评分**：
- **R (Reach)**：影响用户数（1-5，5为全部用户）
- **I (Impact)**：对目标的影响程度（1-5，5为决定性影响）
- **C (Confidence)**：信心系数（0.5-1.0，1.0为高信心）
- **E (Effort)**：开发工作量（人周，数值越小越轻量）
- **RICE Score** = (R × I × C) / E

## 5. 明确不做的功能（Won't Have）

| 功能 | 不做原因 |
|------|----------|
| **独立APP** | MVP阶段聚焦微信小程序，降低获客成本和使用门槛；APP开发维护成本高，与小程序定位重叠 |
| **AI学伴/聊天机器人** | 5-7岁儿童文字交互能力有限，语音对话体验不成熟，投入产出比低 |

| **社交功能（社区/广场）** | 儿童在线社交有安全风险和合规要求，MVP阶段不做 |
| **全学科拓展（数学/英语/拼音）** | MVP聚焦识字单一场景做深做透，避免功能蔓延 |
| **UGC内容创作** | 内容质量不可控，审核成本高，MVP阶段全部使用专业制作内容 |
| **实时视频/直播课** | 与"轻量小程序游戏"定位冲突，商业模式不同 |

| **写字板/硬件配套** | 增加硬件供应链复杂度，MVP用屏幕书空替代 |

## 6. 用户故事

### 故事1：闯关识字（核心流程）

> **作为**一个6岁的孩子，  
> **我想**在冒险岛上闯关，每过一关学会几个新字，  
> **以便**我能在玩的过程中认识更多字，解锁更多关卡和角色。

**验收要点**：每个关卡学习3-5个新字，通过认→图→读→写→测五步完成，通过测验解锁下一关。

### 故事2：家长查看学习效果

> **作为**一个焦虑的妈妈，  
> **我想**打开家长面板就能看到孩子今天学了哪些字、掌握率多少、哪些字还不熟，  
> **以便**我能了解孩子的真实学习进度，而不是只看到"已识字500"这种虚的数字。

**验收要点**：家长面板展示今日学习字列表、每字掌握状态（已掌握/待复习/薄弱）、累计掌握率。

### 故事3：防沉迷管控

> **作为**一个担心孩子沉迷手机的妈妈，  
> **我想**设置每天最多学15分钟，到时间自动停，  
> **以便**孩子不会过度使用手机，也不会沉迷游戏环节。

**验收要点**：家长可设置每日时长（5/10/15/20分钟），到时弹出提醒并锁定新内容，只允许复习。

### 故事4：同步学校生字

> **作为**一个想配合学校教学的家长，  
> **我想**把学校课本里的生字添加到孩子的学习列表中，  
> **以便**孩子学的字和学校教的一致，不浪费精力。

**验收要点**：家长面板支持手动输入/选择生字，系统自动生成识字卡片并插入学习队列。

### 故事5：科学复习不遗忘

> **作为**一个注重科学育儿的爸爸，  
> **我想**系统自动安排复习，按照遗忘曲线在孩子快要忘记时提醒复习，  
> **以便**孩子学过的字不会忘，真正掌握。

**验收要点**：基于间隔复习算法（1天/2天/4天/7天/15天），自动生成每日复习任务，复习完成更新该字掌握状态。

### 故事6：免费试用后透明购买

> **作为**一个被自动续费坑过的家长，  
> **我想**先免费试用，觉得好了一次性买断，永远不会被偷偷扣费，  
> **以便**我放心使用，不用担心被套路。

**验收要点**：免费学50字无限制使用；付费页面明码标价一次性买断；无任何自动续费机制。

### 故事7：收集成就保持兴趣

> **作为**一个喜欢收集的孩子，  
> **我想**每过一关都能获得星星和徽章，集齐徽章能解锁新角色，  
> **以便**我有动力继续学下去。

**验收要点**：每关3星评价（认字正确率/跟读完成度/测验通过率），徽章按识字里程碑发放（50字/100字/200字...），角色皮肤可解锁。

## 8. 商业模式建议

### 8.1 推荐模式：免费增值 + 一次性买断

| 模式要素 | 方案 |
|----------|------|
| **免费部分** | 前50个高频字完全免费（含全部功能：闯关、复习、家长面板），让家长和孩子充分体验 |
| **付费方式** | 一次性买断解锁全部300-500字课程，永不续费 |
| **建议定价** | **¥68-98 一次性买断**（MVP阶段低价获客，后续可调） |
| **不做的模式** | ❌ 月度/年度订阅 ❌ 自动续费 ❌ 分板块二次收费 ❌ 广告 |

### 8.2 定价依据

| 对比项 | 洪恩识字 | 叫叫识字 | 帮帮识字 | 本产品 |
|--------|----------|----------|----------|--------|
| 永久包 | 248-388元 | 198-388元 | 298元 | **68-98元** |
| 订阅 | 18元/月 | — | 19元/月 | **无** |
| 免费部分 | 20字 | 部分内容 | 少量试用 | **50字** |
| 自动续费 | 有（差评严重） | 有 | 有 | **无** |

### 8.3 选择一次性买断的理由

1. **市场痛点驱动**：自动续费是行业最大差评来源（洪恩57个月、宝宝巴士3年），透明收费是核心差异化卖点
2. **家长心理**：一次性买断"心理踏实"，订阅制"随时可退"但实际容易忘记取消
3. **小程序特性**：微信小程序天然不适合复杂订阅管理，买断制流程简单
4. **MVP策略**：低价买断快速获客，建立口碑后再考虑增值服务
5. **内容成本**：300-500字内容制作成本可控，买断制可覆盖

### 8.4 未来增值方向（P2阶段考虑）

| 增值项 | 说明 |
|--------|------|
| 分级阅读绘本包 | 学完基础字后，付费解锁更多绘本 |
| 进阶字包（500-1000字） | 基础包学完后，付费扩展更多字 |
| 实体字卡周边 | 线上线下结合，购买实体识字卡 |
