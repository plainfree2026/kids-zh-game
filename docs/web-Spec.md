# Spec - 网页版识字游戏 v1.0.0

> 生成日期：2026-07-18
> 基于：web-PRD v1.0 + web-Architecture v1.0 + web-UIUX v1.0
> 状态：已确认

---

## 1. 产品定义
- **一句话描述**：6 岁儿童网页识字游戏，含「听音选字测评」+「抓字小游戏」，覆盖常用 3000 字，单 HTML 离线即玩、主打有趣。
- **目标用户**：6 岁幼小衔接 / 小学低年级儿童，主用手机浏览器（家长辅助）。
- **核心问题**：市面识字产品多为 App 需安装/付费，缺免费即玩、零门槛的网页趣味识字。

## 2. MVP 范围（锁定——不在此列表的功能一律不做）

### 明确不做（Won't Have）
- 真实付费 / 账号系统（纯前端单文件，无后端）
- 书写描摹 / 笔顺动画（本期聚焦「测评 + 游戏」两件有趣的事）
- 联网排行榜 / 多人

## 3. 技术架构（锁定）
- **形态**：单个 `.html` 文件，零后端、零构建依赖，双击即玩、可分享。
- **拼音库**：构建期内联 3000 字 `{char: pinyin}` 到 `window.DICT`，离线可用。
- **发音**：Web Speech API `speechSynthesis`（lang=zh-CN）；含 iOS Safari 需用户手势触发的兼容处理。
- **状态**：`localStorage`（key=`shizi_progress_v1`）。
- **响应式**：mobile-first，flex + vw/vh，触屏优先，按钮 ≥ 44px。

## 4. 模块清单（单文件内 JS 模块）

| 模块 | 职责 |
|------|------|
| data | DICT 字典读取 + 按级别取样 / 干扰字生成 |
| speech | 发音封装（UA 兼容、队列、降级） |
| assess | 测评流程（出题/判分/估算） |
| game | 抓字游戏循环（spawn/下落/点击/命/连击） |
| mascot | 字宝表情与正向反馈文案 |
| storage | 进度读写 |
| ui | 视图切换与渲染 |

## 5. 数据库表清单（锁定）
> 无后端。本地存储结构：
```
shizi_progress_v1 = {
  highestLevel: 1,        // 已解锁最高级别
  bestScore: 0,           // 抓字游戏最高分
  bestCombo: 0,           // 最高连击
  assessedLevel: 0,       // 最近测评建议级别
  lastAssess: null        // 最近测评时间
}
```

## 6. 数据（3000 字库）
- **来源**：《通用规范汉字表》一级字前 3000（按规范频序）。
- **结构**：`{ "我":"wǒ", "你":"nǐ", ... }`。
- **分级**：按索引分 30 级，每级 100 字，难度递增。另提供「按年级分」类别：按人教版小学《识字表》各册识字量切分 12 册（一年级上…六年级下），字序简单字在前，近似年级递进。
- **测评取样**：从 `已学级别 ~ 已学级别+1` 随机抽 20 字，每字配 3 个干扰字。
- **游戏取样**：当前级别字为主，混入 1~2 个其他级干扰字。

## 9. 验收标准（锁定——QA 测试时以此为唯一依据）

| 编号 | 功能 | Given | When | Then |
|------|------|-------|------|------|
| AC01 | 测评 | 进入测评 | 播 20 题拼音、每题 4 选 1 | 结束显示识字量估算 + 建议级别，存 assessedLevel |
| AC02 | 抓字游戏 | 进入游戏 | 字从顶部下落 + 听音，点中正确字 | 加分+连击；漏接/错点扣 1 命；3 命尽结束出分 |
| AC03 | 发音 | 任意字需发音 | 调用 speech.speak(字/拼音) | 用 speechSynthesis 播标准音；iOS 用户手势可触发；不支持时降级视觉提示不崩 |
| AC04 | 离线字库 | 断网 | 打开页面 | 3000 字拼音可用，测评/游戏可玩 |
| AC05 | 趣味反馈 | 答对/连击 | 触发反馈 | 字宝变表情 + Confetti + 语音「真棒」；连击有缩放特效 |
| AC06 | 进度保存 | 玩过一局 | 刷新页面 | highestLevel/bestScore/bestCombo/assessedLevel 不丢 |
| AC07 | 视图导航 | 任意视图 | 点导航 | 5 视图互相可达，无死路 |
| AC08 | 级别递进 | 解锁低级别 | 进高级别 | 字难度递增；未解锁级别不可进；支持字频/年级/拼音三类别 |
| AC09 | 抓拼音游戏 | 进入抓拼音 | 汉字下落+听音，点中正确拼音 | 加分+连击；漏接/错点扣 1 命；3 命尽结束出分 |
| AC10 | 单局时长 | 进入抓字/抓拼音 | 开始游戏 | 顶部显示 60 秒倒计时，归零即结束并出分 |

## 10. 边界与约束
- 不支持 IE；需现代浏览器（iOS 11+ / Android Chrome）。
- `speechSynthesis` 不支持时降级为视觉高亮 + 文字拼音提示，不报错崩溃。
- 单文件体积 < 5MB。
- 触屏按钮 ≥ 44px；防误触（点击 300ms 节流）。
