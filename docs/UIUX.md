## 1. 设计理念与对标品牌

### 1.1 设计理念

本产品面向5-7岁幼小衔接阶段儿童，核心设计理念为：**"温暖陪伴，趣味探索，润物无声"**。

基于2025年儿童数字产品设计的核心趋势——儿童倾向于**平静、可预测、有结构**的体验，而非过度刺激的快节奏内容（来源：2025年Kids Digital Play洞察报告）。我们遵循以下设计原则：

- **适度刺激，而非过度刺激**：色彩明亮但不刺眼，动效轻快但不闪烁，避免感官过载
- **60-30-10配色法则**：主色60%、辅助色30%、强调色10%，控制色彩数量在3种主色以内，提升儿童操作正确率（来源：AI幼教产品FRIEND框架研究，操作正确率提升至92%）
- **三秒原则**：任何核心交互都应在3秒内完成，减少认知负荷
- **隐形教学**：将学习目标融入游戏化场景，让儿童在"玩"中自然习得汉字
- **具象化设计**：图标和视觉元素使用具象、直觉性的图形，避免抽象符号

### 1.2 竞品分析

| 竞品 | 视觉风格 | 优势 | 不足 |
|------|----------|------|------|
| **洪恩识字** | 3D卡通、糖果色、清新明亮 | 画风精致，UI简洁，"玩-认-练-写-测"五维闭环，粒子笔顺特效 | 3D制作成本高，小程序难以复刻 |
| **悟空识字** | 西游记IP、色彩浓重、传统卡通 | 故事性强，IP贯穿，剧情连贯 | 配色过重，特效较旧，PPT既视感，女童吸引力不足 |
| **叫叫识字** | 小鸡叫叫IP、明快活泼、冒险故事 | 2025红点奖，配色科学，关键内容居中放大动态高亮，激励机制可视化（连胜火焰） | 以阅读为核心，识字为辅 |
| **宝宝巴士** | 高饱和色彩、简约卡通、可爱软萌 | 全球8亿用户验证，高饱和互动色+柔和叙事色双系统，适合低龄 | 节奏偏慢，对5-7岁稍显幼稚 |

### 1.3 对标品牌与设计方向

**主对标品牌：洪恩识字 + 叫叫识字**

选择理由：
1. **洪恩识字**的简洁UI布局和科学分级体系是行业标杆，其"玩-认-练-写-测"闭环适合我们的闯关认字玩法
2. **叫叫识字**的IP角色驱动和可视化激励机制（2025红点奖获奖设计）值得借鉴，其"关键内容居中放大+动态高亮"的界面体系经儿童认知心理学验证

**设计语言定位：2D扁平卡通 + 温暖糖果色系**

- 采用2D扁平插画风格（而非3D），降低小程序制作成本，同时保证视觉精致度
- 色彩参考洪恩识字的糖果色体系，但降低饱和度，避免过度刺激
- 引入吉祥物IP角色（参考叫叫的小鸡叫叫模式），构建情感化陪伴系统
- 关键学习内容居中放大，配合动态高亮，方便儿童辨识

## 2. 配色方案

### 2.1 配色策略

采用**"温暖糖果色"**体系，遵循60-30-10法则：
- 主色调（60%）：温暖橙色——代表活力、好奇心和成就感
- 辅助色（30%）：天蓝色 + 嫩绿色——代表探索、信任和成长
- 强调色（10%）：明黄色——用于关键操作引导和奖励反馈

设计依据：
- 研究表明3-4岁儿童对高饱和度色彩识别度是低饱和度的2.8倍，但超过3种主色会导致注意力分散
- 2025年趋势显示，儿童产品正向"温和明亮"而非"高饱和刺激"方向演进
- 暖色调（橙/黄）在中国文化中关联积极性和好运，适合教育产品

## 3. 字体选择

### 3.1 字体选择原则

- **圆润友好**：笔画边缘柔和，无尖锐棱角，给儿童亲切感
- **清晰易读**：字形结构规范，适合刚识字的儿童辨认
- **免费可商用**：降低项目成本，便于小程序嵌入
- **层级分明**：标题与正文有足够区分度

### 3.2 中文字体方案

| 字体角色 | 字体名称 | 字重 | 使用场景 | 授权 |
|----------|----------|------|----------|------|
| 显示字体（标题） | **阿里妈妈方圆体** | Bold / Heavy | 页面大标题、关卡名称、奖励弹窗 | 免费商用 |
| 正文字体 | **阿里巴巴普惠体** | Regular / Medium | 汉字卡片、正文内容、按钮文字、说明文案 | 免费商用 |
| 汉字教学字体 | **阿里妈妈东方大楷** | Regular | 汉字笔顺演示、字源讲解、书写范本 | 免费商用 |

**选择理由**：
- **阿里妈妈方圆体**：笔画边缘经过柔和处理，无尖锐棱角，整体亲切温暖活泼，官方推荐场景明确包含"儿童产品与教育"（来源：阿里字体官方介绍）
- **阿里巴巴普惠体**：无衬线字体，笔画粗细适中，字形方正不失圆润，在手机屏幕上阅读舒适度极佳，已被大量App采用
- **阿里妈妈东方大楷**：继承传统楷体精髓，针对数字化阅读做了优化，适合汉字教学场景的规范书写展示

### 3.3 英文/数字字体方案

| 字体角色 | 字体名称 | 字重 | 使用场景 |
|----------|----------|------|----------|
| 英文标题 | **Fredoka** | SemiBold | 英文标题、数字徽章 |
| 英文正文 | **Nunito** | Regular / Bold | 英文内容、数字 |
| 数字强调 | **Fredoka** | Bold | 分数、星星数量、倒计时 |

### 3.4 字号规范

| 层级 | 字号 | 行高 | 使用场景 |
|------|------|------|----------|
| Display | 48rpx | 1.2 | 奖励弹窗大标题、过关庆祝 |

## 4. 图标风格方向

### 4.1 图标设计原则

- **具象化优先**：使用儿童能直觉理解的具象图形，而非抽象符号。例如用"画笔"代表绘画，用"背包"代表任务，用"星星"代表奖励
- **圆润饱满**：所有图标采用圆润边角，无尖锐棱角，与整体圆润设计语言一致
- **大尺寸**：功能图标最小尺寸48x48rpx，核心操作图标64x64rpx，确保儿童手指可准确触达
- **填充式风格**：优先使用填充式（Filled）图标而非线性（Outlined）图标，填充式图标对儿童更友好、辨识度更高
- **不用emoji作为功能图标**：emoji在不同设备渲染不一致，且不够专业

### 4.2 图标风格定义

**风格：2D扁平填充 + 圆润边角 + 微渐变**

- 基础形状：圆角矩形（border-radius ≥ 12rpx）
- 填充方式：实色填充或微渐变（同色系2色渐变）
- 描边：无描边或1px同色系深色描边
- 阴影：可选，柔和投影（0 4rpx 8rpx rgba(0,0,0,0.08)）

### 4.3 推荐图标库

| 图标库 | 说明 | 适用场景 |
|--------|------|----------|
| **iconfont（阿里图标库）** | 国内最大免费图标库，支持中文搜索，可自定义颜色和大小 | 功能图标、UI图标 |
| **Phosphor Icons** | 开源图标库，提供Fill/Regular等多种风格，圆润友好 | 系统级图标 |
| **自绘图标** | 针对核心功能（如关卡地图、识字卡片操作）定制绘制 | 核心玩法相关图标 |

## 5. 插画/角色设计方向

### 5.1 吉祥物设计

**吉祥物名称：字宝（暂定）**

**设计方向**：
- **物种**：拟人化小动物，推荐**小狐狸**或**小熊猫**——狐狸代表聪慧，小熊猫代表珍稀可爱
- **比例**：大头小身（头身比约1:1.5），符合儿童审美偏好（来源：儿童角色设计研究，大头小身比例更受儿童喜爱）
- **色彩**：主色调与品牌色呼应——若选小狐狸，毛色为暖橙色（#FF8C42系）；若选小熊猫，毛色为红棕色+白色面部
- **表情**：默认表情为大眼微笑，设计多套表情：开心、思考、惊喜、鼓励、加油
- **配件**：头戴一顶小帽子（如探险帽），暗示"冒险闯关"主题
- **动态**：设计待机、跳跃、欢呼、指路等基础动作

**吉祥物使用场景**：
- 首页顶部迎接儿童
- 关卡地图上作为角色行走标记
- 识字卡片页作为引导者
- 答对/答错时给出表情反馈
- 奖励弹窗中一起庆祝

### 5.2 插画风格

**风格定义：2D扁平卡通 + 温暖手绘感**

- **线条**：粗细均匀的圆润线条，无尖角，线条颜色为深棕（#5D4037）而非纯黑，更柔和
- **填色**：扁平实色填充为主，关键元素可使用同色系微渐变增加层次
- **质感**：可添加微妙的噪点纹理，增加手绘温暖感
- **背景**：场景插画使用柔和的渐变背景，配以简单的装饰元素（云朵、树木、小山丘）
- **人物**：所有人物角色遵循大头小身比例，表情丰富夸张

### 5.3 场景插画规划

| 场景 | 描述 | 出现位置 |
|------|------|----------|
| 冒险地图 | 蜿蜒小路连接不同关卡，沿途有树木、小山、河流 | 关卡地图页背景 |
| 森林营地 | 营地帐篷、篝火、旗帜 | 首页背景/章节切换 |
| 汉字乐园 | 由汉字元素组成的奇幻场景 | 识字卡片页背景 |
| 庆祝舞台 | 颁奖台、彩带、礼花 | 过关奖励弹窗 |
| 成长花园 | 花朵、小树苗随学习进度成长 | 学习报告页 |

### 5.4 插画制作建议

- 矢量绘制（SVG/Figma/AI），保证小程序内缩放不失真
- 每张插画导出PNG和SVG两种格式，SVG优先用于小程序
- 建立插画资产库，统一风格指南
- 关键场景委托专业插画师绘制，日常元素使用组件化拼搭

### 6.2 各页面详细规划

#### 6.2.1 首页

**核心功能**：入口聚合，快速进入学习

**核心元素**：
- 顶部：吉祥物"字宝"迎接动画 + 用户头像/昵称
- 中部：大尺寸"继续学习"按钮（显示当前进度，如"第3关 水果乐园"）
- 中下：今日学习目标卡片（已学/目标字数，星星进度条）
- 底部：功能入口区——"关卡地图"、"奖励中心"、"学习报告"
- 底部Tab栏：首页、地图、报告、我的

**设计要点**：
- "继续学习"按钮占屏幕宽度70%以上，高度≥96rpx
- 吉祥物有呼吸动画，增加活力感
- 背景使用淡渐变（暖奶白→浅橙）

#### 6.2.2 关卡地图页

**核心功能**：展示学习路径，选择关卡

**核心元素**：
- 全屏冒险地图背景插画
- 蜿蜒路径串联关卡节点
- 已通关关卡：显示星星数（1-3星），可点击重玩
- 当前关卡：高亮闪烁，吉祥物站立旁

- 章节标题在顶部（如"第一篇：自然万物"）

**设计要点**：
- 地图可纵向滚动
- 关卡节点为圆形按钮，直径≥80rpx
- 当前关卡有脉冲动画引导注意

#### 6.2.3 识字卡片页（核心）

**核心元素**：

**设计要点**：
- 卡片有翻转/滑动切换动画
- 汉字使用阿里妈妈东方大楷，规范楷体

## 7. 交互设计原则

### 7.1 触摸区域规范

根据儿童手指精细动作发育特点（5-7岁儿童手指控制力弱于成人），制定以下规范：

| 元素类型 | 最小触摸区域 | 推荐尺寸 | 间距要求 |
|----------|-------------|----------|----------|
| 主操作按钮 | 88x88rpx | 96x96rpx | - |
| 次级按钮 | 72x72rpx | 80x80rpx | - |
| 功能图标 | 48x48rpx | 64x64rpx | - |
| 选项卡片 | 120x120rpx | 140x140rpx | ≥24rpx |
| 列表项 | - | 高度≥80rpx | - |
| 按钮间距 | - | - | ≥24rpx |
| 图标间距 | - | - | ≥16rpx |

> **设计红线**：交互元素间距≥24rpx（物理尺寸约12mm），防止误触。来源：AI幼教产品友好性设计指南。

### 7.2 防误触设计

- **按钮间距**：所有可点击元素间距≥24rpx
- **二次确认**：退出学习环节、返回首页时弹出确认弹窗（大按钮"继续学习"/"确定退出"）
- **误触容忍**：练习环节选错不扣分不惩罚，鼓励试错
- **防连续点击**：按钮点击后300ms内禁用重复点击，防止快速连点
- **安全区域**：关键操作按钮避开屏幕边缘32rpx以内区域
- **家长锁**：家长中心入口需长按3秒或解答算术题

### 7.3 反馈动效设计

| 场景 | 动效类型 | 时长 | 说明 |
|------|----------|------|------|
| 按钮点击 | 缩放+颜色变化 | 150ms | scale(0.95) + 颜色加深 |
| 答对反馈 | 星星弹跳+粒子 | 1200ms | 星星从中心弹出+碎屑粒子效果 |
| 答错反馈 | 卡片轻微摇晃 | 400ms | 摇晃幅度小，配合鼓励语音 |
| 卡片切换 | 滑入滑出 | 300ms | 左滑出+右滑入，ease-out |
| 关卡解锁 | 光效+缩放 | 800ms | 金色光效扩散+关卡按钮放大 |
| 过关庆祝 | 彩带+吉祥物跳跃 | 2000ms | 全屏彩带飘落+吉祥物欢呼 |
| 进度条填充 | 宽度渐变增长 | 600ms | ease-out，配合数字滚动 |
| 页面转场 | 淡入淡出 | 250ms | 简洁过渡，不花哨 |

> **设计红线**：
> - 操作反馈时长≤500ms（来源：AI幼教产品友好性设计指南）
> - 避免闪烁效果，可能引发光敏反应
> - 动效节奏轻快有节制，不使用晃动、旋转等可能引起不适的效果

### 7.4 导航方式

- **底部Tab栏**：固定底部，4个主入口（首页、地图、报告、我的），图标+文字
- **返回按钮**：左上角固定位置，所有子页面都有
- **面包屑**：不使用（儿童不理解），改用页面标题+返回按钮
- **不使用隐藏菜单**：不使用汉堡菜单等隐藏导航，所有功能入口可见
- **导航层级**：不超过2级，核心功能3步内可达

### 7.5 声音设计原则

- 所有操作配有音效反馈（点击、成功、失败、过关）
- 答对音效：欢快上扬音阶
- 答错音效：柔和提示音（非刺耳蜂鸣），配合鼓励语音
- 背景音乐：轻快但不干扰，可在设置中关闭
- 汉字发音：标准普通话女声，语速适中偏慢

### 7.6 无障碍设计

- 文字与背景对比度≥4.5:1（WCAG AA标准）
- 所有交互元素提供alt文本
- 支持系统字体放大
- 色彩不仅作为唯一信息区分手段（如错误状态除颜色外加图标）
- 避免纯色彩编码，辅以文字/图标说明
