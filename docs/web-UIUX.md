# 网页版识字游戏 · UIUX（设计文档）

> 版本：v0.1（待用户确认）
> 日期：2026-07-18
> 设计语言：2D 扁平卡通 + 温暖糖果色系（与小程序版一脉相承，但更"轻、快、蹦")

---

## 1. 设计 Token（锁定）

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 暖橙 | `#FF8C42` | 主按钮、得分、吉祥物 |
| 辅助 薄荷蓝 | `#4ECDC4` | 候选字、正确反馈 |
| 辅助 嫩芽绿 | `#95E1A3` | 连击、成功 |
| 强调 明黄 | `#FFD93D` | 星星、高亮 |
| 背景 暖奶白 | `#FFF9F0` | 页面底色 |
| 文字 | `#3D3A36` | 正文 |
| 文字柔 | `#8A837A` | 次要信息 |

- **字体**：系统圆体优先（`PingFang SC` / `Microsoft YaHei`）；标题可用更圆的 web 字体（可选 CDN，非必需）
- **圆角**：大圆角（16–24px），按钮果冻感
- **吉祥物**：字宝 🦊（沿用小程序 IP），表情随连击变化（普通→开心→得意）

## 2. 信息架构（5 个视图，单页切换）

```
[首页]           字宝 + 大字标题 + 两个大按钮(开始测评 / 开始游戏) + 星星展示
  │
  ├─[测评]        听音题目 + 4 候选 + 进度条
  ├─[报告]        识字量大字 + 建议级别 + 鼓励语 + 入口
  │
  └─[游戏]        下落区 + 发音引导 + 底部 4 候选 + 顶部 HUD(分数/生命/连击)
     └─[结算]     本局得分 + 获得星星 + 最高连击 + 重玩/回首页
```

## 3. 各视图设计要点

### 3.1 首页
- 顶部字宝 🦊 蹦跳欢迎：「嗨，我是字宝！今天认几个字？」
- 中央两个超大圆角按钮：**开始测评**（薄荷蓝）、**开始游戏**（暖橙）
- 下方小字：⭐ 星星数 · 已识 X 字
- 背景：暖奶白 + 漂浮的半透明汉字装饰（缓慢上浮，增加"字海"氛围）

### 3.2 测评页（听音选字）
- 顶部：进度条「3 / 20」+ 重听按钮 🔊
- 中部：留白（不直接显示字，强制"听音"）
- 底部：2×2 四个大候选字卡（圆角、点击有果冻下陷）
- 点正确：绿光扩散 + 字宝比心 + 短促鼓励音
- 点错误：红框抖动 + 自动重播正确读音 + 字宝「再想想~」

### 3.3 报告页
- 大字「你认识了约 **X** 个字！」+ 星星雨 Confetti
- 建议：「建议从 **第 Y 级** 开始玩 🎮」
- 两个入口按钮：再测一次 / 去玩游戏

### 3.4 游戏页（抓字）
- 顶部 HUD：❤️×3（生命）· ⭐分数 · 🔥连击数
- 中部下落区：汉字以圆角气泡从顶部随机位置下落，目标字出现时字宝在侧边提示发音
- 底部：4 个候选字卡（含当前目标字）
- 点中正确：该字"啪"爆炸成星星粒子 + 连击 +1 + 分数跳变
- 漏接/点错：屏幕轻震 + 生命 -1 + 字宝「哎呀」
- 连击特效：🔥 连击 ≥5 时下落加速、候选卡描边发光、背景轻微缩放脉冲

### 3.5 结算页
- 居中：本局得分大数字（弹入）+ ⭐ 获得星星数 + 🔥 最高连击
- 字宝庆祝动画 + Confetti
- 按钮：再玩一局（暖橙）/ 回首页（幽灵按钮）

## 4. 趣味要素清单（"有趣"的落地）

1. **果冻动效**：所有按钮/卡片点击有下陷回弹（cubic-bezier 弹性曲线）
2. **Confetti 粒子**：答对、结算、报告页撒星星/爱心
3. **连击系统**：连击越高反馈越炫（音效升调 + 视觉加速），制造"上头"感
4. **吉祥物情绪**：字宝随连击/成败切换表情与台词（普通/开心/得意/安慰）
5. **正向声音**：答对用 speechSynthesis 读"真棒！/ 厉害！"等随机鼓励语（中文语音，无需音频文件）
6. **漂浮字海背景**：首页/游戏背景缓慢上浮的半透明汉字，强化"识字"主题且不干扰
7. **即时反馈**：任何操作 100ms 内给出视觉+（可选）听觉反馈，6 岁娃不等待不迷茫

## 5. 反模式检查（自检）

- ✅ 无紫色渐变、无滥用 emoji 图标（吉祥物 emoji 为有意 IP 表达，允许）
- ✅ 无千篇一律 Hero 大图（用吉祥物+动效替代）
- ✅ 大字号、大点击区（候选卡 ≥ 72px，适配儿童手指）
- ✅ 高对比、暖色调，护眼不刺眼
- ✅ 无误导按钮、无诱导分享

## 6. 响应式

- 容器 `max-width: 460px` 居中，手机竖屏天然适配
- 桌面/平板：容器限宽，留白两侧；触屏与鼠标统一用 `click`
- 安全区：`env(safe-area-inset-*)` 适配刘海屏
